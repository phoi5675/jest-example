# Expressjs with jest

A simple example of how to use Jest testing framework with ExpressJS.

## Prerequisites

- node.js (>= v20)
- npm

eslint and prettier are also included in the project for code linting and
formatting.

## Installation

```bash
npm install
```

## Run

```bash
npm run start
```

## Test

```bash
npm run test
```
